## About

This repository contains my personal study materials for **Intermediate Microeconomics**.

The notes were written while I was taking the course at **NCCU**. They combine formal lecture content, in-class explanations, worked examples, derivations, selected textbook materials, and my own interpretations of the course topics.

The purpose of this repository is to organize my learning process and provide a reference for students who are studying similar topics.

These materials may be helpful if you are:

* Taking Intermediate Microeconomics
* Reviewing microeconomic theory
* Preparing for exams
* Studying with Perloff's textbook
* Preparing for economics-related graduate entrance exams
* Looking for additional explanations or worked examples

Please note that this is a personal note collection, not an official course website.

## How to Use These Notes

These notes are intended to be used as a **study aid**.

A recommended way to use this repository is:

* Start with the lecture notes to understand the main concepts
* Use the selected chapter notes to review textbook-related materials
* Practice with the problem sets after reviewing each topic
* Use the graduate entrance exam materials for additional practice
* Compare the notes with the textbook and official course materials when possible

These notes work best when used alongside lectures, textbooks, and problem-solving practice.
